# Noema

Сервис для создания миров: бестиарии, таймлайны, вики-разделы, карты и многое другое.
Подходит для работы писателям, геймдизайнерам, сценаристам и людям иных творческих профессий.

## Модули сервиса

Сервис включает ряд самодостаточных модулей для агрегации информации о созданных Мирах:
- Мировые дашборды: каждый созданный мир имеет свои базовые параметры (Название, Инициирующее событие, Краткий синопсис, Изображение (опционально)). 
- Таймлайн: сервис по визуализации событий от Инициирующего события (Большой Взрыв, Творение Вселенной, утро понедельника). На один Мир приходится один Таймлайн, но в рамках него можно делать разные временные линии для разных событий, героев и т.д.
- Карточки: сервис для написания сценария/произведения карточным методом. На один Мир может приходиться много Историй в карточках. При создании новой Истории, создаётся 4 карточки, соответствующие четырёхактной структуре. В каждой карточке можно писать сцены по логике 1 абзац = 1 сцена, далее карточку можно декомпозировать (5 абзацев карточки превращаются в 5 карточек) для создания Истории от общего к частному.
- Доски взаимосвязей: сервис для создания взаимосвязей по аналогии с детективными романами. Можно производить и комментировать взаимосвязи между разными сущностями - событиями таймлайнов, бестиариями, биографиями и пр.
- Бестиарии: сервис для создания классического бестиария созданного Мира.
- Биографии: сервис для создания Библий персонажей.

## Технологии

- Laravel 12
- PHP 8.2+
- SQLite (по умолчанию)

## Установка

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm install
npm run build
```

## Запуск

```bash
php artisan serve
```

Приложение будет доступно по адресу http://localhost:8000
